KeyDragZoom Examples

API Reference

Include Scripts

This class allows a user to zoom in to an area on the map by holding down a hot key (e.g., Shift) while dragging a box around the area. It works with Google Maps API V3.

The first step is to include keydragzoom.js or keydragzoom_packed.js in your document header, after the Google Maps API is loaded. You can use the online version if you do not want to download the script.

      <script src="/path/to/keydragzoom_packed.js" type="text/javascript"></script>

Enable KeyDragZoom

You can simply call google.maps.Map.enableKeyDragZoom() to enable KeyDragZoom.

        function init(){
         var myOptions = { 
            zoom: 13, 
            center: new google.maps.LatLng(49.2903, -123.1294), 
            mapTypeId: google.maps.MapTypeId.ROADMAP 
           };
          var map = new google.maps.Map(document.getElementById("map"), myOptions); 
          //...
          map.enableKeyDragZoom();  
        }

Styling the Zoom Box

You can pass some CSS properties when you enable KeyDragZoom. Properties for boxStyle will be applied to the zoom box and properties for paneStyle will be applied to the "veil" that appears over the map when you hold down the hot key. In the following example, the zoom box is 50% transparent with a red background and a thick dashed black border.

        function init(){
          var myOptions = { 
            zoom: 13, 
            center: new google.maps.LatLng(49.2903, -123.1294), 
            mapTypeId: google.maps.MapTypeId.ROADMAP 
           };
          var map = new google.maps.Map(document.getElementById("map"), myOptions); 
          map.enableKeyDragZoom({
                boxStyle: {
                  border: "thick dashed black",
                  backgroundColor: "red",
                  opacity: 0.5
                },
                paneStyle: {
                  backgroundColor: "gray",
                  opacity: 0.2
                }
          });
        }

Defining the Hot Key

If you do not want using the Shift key as the hot key, you can assign the key property to one of the other two keys that are supported: Alt or Ctrl. In the following example, you can start the drag zoom by holding down the Ctrl key

        function init(){
          var myOptions = { 
            zoom: 13, 
            center: new google.maps.LatLng(49.2903, -123.1294), 
            mapTypeId: google.maps.MapTypeId.ROADMAP 
           };
          var map = new google.maps.Map(document.getElementById("map"), myOptions); 
          map.enableKeyDragZoom({
            key: 'ctrl'
          });
        }

Supporting Multiple Maps on One Page

Each map can have its own KeyDragZoom tool that works independently of the other.

        var map1;
        var map2;
            function createMap(mapDiv) {
               var myOptions = {
                zoom: 12,
                center: new google.maps.LatLng(35.227, -80.84),
                mapTypeId: google.maps.MapTypeId.ROADMAP
              }
              var map = new google.maps.Map(document.getElementById(mapDiv), myOptions);
              
              
              map.enableKeyDragZoom({
                key: "shift",
                boxStyle: {
                  border: "2px dashed black",
                  backgroundColor: "red",
                  opacity: 0.5
                },
                paneStyle: {
                  backgroundColor: "gray",
                  opacity: 0.2
                }
              });
              return map;
            }
            
            function init() {
              map1 = createMap('map1');
              map2 = createMap('map2');
              
            }

Events

The KeyDragZoom tool triggers different events at key moments of the drag zoom operation.

       map.enableKeyDragZoom();
          var dz = map.getDragZoomObject();
          google.maps.event.addListener(dz, 'activate', function() {
           // do things
          });
          google.maps.event.addListener(dz, 'dragstart', function() {
           // do things
          });
          google.maps.event.addListener(dz, 'drag', function() {
          // do things
          });
        }
